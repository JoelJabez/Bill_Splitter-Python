Write a program that reads a single line containing key-value pairs as input, where each key-value pair is of the form key:value. The key-value pairs are separated by a space. Create a dictionary from the input key-value pairs and print the dictionary object.
Sample Input:
name:John age:30 city:New York

Sample Output:
{'name': 'John', 'age': '30', 'city': 'New York'}


Sample Input:
fruit:apple quantity:5 price:1.50

Sample Output:
{'fruit': 'apple', 'quantity': '5', 'price': '1.50'}



Memory limit: 256 MB
Time limit: 15 seconds
